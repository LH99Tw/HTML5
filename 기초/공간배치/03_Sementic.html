Sementic 태그

HTML5에서 추가된 시멘틱 태그는 브라우저, 개발자, 검색 엔진에게 웹 페이지의 구조를 더 명확하게 설명할 수 있도록 돕는 태그이다. 주의할 점은 header 태그를 쓴다고 해서 레이아웃을 만들어주지는 않는다. 단지, 의미를 부여할 뿐 레이아웃은 CSS3로 작업한다.


HTML5에서 추가된 시멘틱 태그

  • header : HTML 헤더나 section에 헤더를 정의함
  • nav : HTML 메뉴바를 정의함
  • main : HTML 전체 Content를 정의함
  • section : HTML Section을 정의함
  • article : HTML 하나의 글을 정의함
  • aside : 웹 페이지에서 배너나 추가적인 문구 등을 정의함
  • footer : HTML Footer를 정의함
