HTML 블록 요소와 인라인 요소

앞서 배웠던 모든 HTML 태그는 블록 요소와 인라인 요소 중 하나에 속한다.

  • 블록 요소 : 줄바꿈이 일어나는 요소
  • 인라인 요소 : 줄바꿈이 일어나지 않는 요소

블록 요소

블록 형태의 속성을 가진 Tag들은 말그대로 네모 블록 모양을 가진다.
항상 새로운 줄에서 시작하며, 해당 줄의 모든 너비를 차지한다.
또한 블록 요소는 블록 요소 안에 블록 요소를 포함할 수 있지만, 인라인 요소는 인라인 요소 안에 블록 요소를 포함할 수 없다.
div 태그 : 블록 요소의 기본 태그

  • 다른 HTML 태그들을 묶어서 그룹화하는 태그
  • 하나로 묶인 다른 HTML 태그에 CSS 스타일을 한번에 적용하는 데 자주 사용한다.

예시


HTML5 & CSS3

HTML5와 CSS3를 이용한 웹 페이지 구조화

  • border : 테두리 속성
  • text-align : 텍스트 정렬 속성

블록 형태의 대표적인 태그 :
div, p, ol, ul, li, h#, pre, blockquote, form, table, hr, header, footer, section, article, aside, details, summary ...

인라인 요소

인라인 요소는 블록 요소와 달리 줄바꿈이 일어나지 않는다.
span 태그 : 인라인 요소의 대표적인 태그

  • 텍스트의 특정 부분을 강조하는데 자주 사용한다.
  • 텍스트의 특정 부분에 CSS 스타일을 적용하기 위해 자주 사용한다.

예시
HTML5 & CSS3를 공부하고 있습니다.
인라인 형태의 대표적인 태그 : span, a, q, b, em, small, strong, sup, sub, input, img, br, ...
