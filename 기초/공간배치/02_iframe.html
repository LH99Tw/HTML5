iframe 태그

HTML iframe이란 inline frame의 약자로, 웹 페이지 내에 웹 페이지를 포함할 수 있는 태그이다.


사용법


iframe src = "웹 페이지 주소 URL" title = "내용">
width, height 속성을 이용해 크기를 조절할 수 있다.

iframe을 이용하여 웹 페이지에 다른 웹 페이지 삽입하기


iframe의 테두리 변경 및 삭제


a tag를 이용하여 iframe에 여러 개의 웹 페이지 삽입하기

name 속성을 이용하면 여러 개의 웹 페이지 삽입이 가능하다.
