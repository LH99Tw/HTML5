input 태그

input 태그는 사용자가 다양한 타입의 데이터를 입력할 수 있는 태그이다.

  1. text : 한줄 텍스트 입력
  2. password : 비밀번호 입력
  3. radio : 여러 항목 중 하나 선택
  4. checkbox : 여러 항목 중 다중 선택
  5. file : 파일 업로드
  6. textarea : 여러줄 텍스트 입력
  7. select : 드롭다운 목록
  8. button : 버튼
  9. submit : 전송 버튼
  10. fieldset : 그룹화
