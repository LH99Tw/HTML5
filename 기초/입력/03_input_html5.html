input 태그

HTML5로 넘어오면서 더 다양한 타입의 input 태그가 추가되었습니다.

  1. color : 색상 선택
  2. date : 날짜 선택
  3. time : 시간 선택
  4. datetime-local : 날짜와 시간 선택
  5. month : 월 선택
  6. week : 주 선택
  7. number : 숫자 입력
  8. range : 숫자 범위 선택
  9. email : 이메일 입력
  10. tel : 전화번호 입력
  11. url : 주소 입력
  12. search : 검색 입력
