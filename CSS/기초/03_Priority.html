03장 중요도(Importance)

중요도는 스타일 시트에서 스타일 규칙의 중요성을 나타내는 개념이다.

중요도는 스타일 시트에서 스타일 규칙의 중요성을 나타내는 개념이다.


위의 예시에서 인라인 스타일 시트, id 선택자, class 선택자, Tag 선택자 모드 쓰였는데, 모든 설정이 같은 경우 우선순위는 인라인 스타일 시트 > id 선택자 > class 선택자 > Tag 선택자 순으로 적용된다.

하지만만, 위의 결과는 초록색의 시트가 적용이 되었다. 그 이유는 !important 속성이 있기 때문이다. 다른 것들을 고려하지 않고 가장 우선적으로 고려하기 때문입니다.

  1. 중요도(importance) : 속성 값 뒤에 !important 속성을 추가하면 가장 우선적으로 고려한다다.
  2. 명시도(specificity) : 모든 설정이 같을 경우 가장 나중에 선언한 것이 우선되어 적용됩니다.
    1. 1. 인라인 스타일 시트
    2. 2. 외부 또는 내부 스타일 시트 : id > class > Tag
    3. 3. 브라우저 기본 값
  3. 브라우저의 기본 값
