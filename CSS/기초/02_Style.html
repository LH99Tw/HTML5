03장-2 스타일(Style) 적용하기기

CSS3를 HTML5에 적용하는 방법은 크게 3가지가 있다.


  • 인라인 스타일
  • 내부부 스타일
  • 외부 스타일

인라인 스타일

인라인 스타일은 HTML 태그에 style 속성을 추가하여 스타일을 적용한다.


내부 스타일

내부 스타일은 head 태그에 style 속성을 추가하여 스타일을 적용한다.


외부 스타일

외부 스타일은 별도의 CSS 파일을 만들어 스타일을 적용한다.

각각 HTML 문서의 head 태그 내에서 link 태그를 이용하여 선언한다.
link rel="stylesheet" type="text/css" href="css 파일명"
